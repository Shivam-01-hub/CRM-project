# Project Notes
+
+This folder captures lightweight documentation for the Partnership CRM app.
+
+## Architecture
+
+- The app is a React + Vite + TypeScript single-page application.
+- CRM data is seeded from `src/data/seed-organizations.json` and stored in `localStorage` for persistence.
+- The main dashboard is intentionally split into smaller components under `src/components/`.
+
